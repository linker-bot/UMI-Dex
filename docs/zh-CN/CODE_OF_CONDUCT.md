# 贡献者公约行为准则

**UMI-Dex** 是由 **Linkerbot** 维护的灵巧手遥操作数据采集栈，当前采用 ROS1 录制与 Python 分析工具并行的架构。本项目采用 **Apache License 2.0** 授权。社区参与遵循 **Contributor Covenant 2.1**（贡献者公约）。

**其它语言版本：**见 [文档索引](../README.md)。

---

### 贡献者公约行为准则

#### 我们的承诺

作为社区成员、贡献者与负责人，我们承诺为每个人提供无骚扰的参与体验，无论其年龄、体型、明显或隐性的残障、族裔、性征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、种姓、肤色、宗教或性取向如何。

我们承诺以有助于营造开放、友善、多元、包容且健康社区的方式行动与互动。

#### 我们的标准

有助于为本社区营造积极环境的示例行为包括：

* 对他人展现同理心与善意
* 尊重不同的观点、立场与经历
* 给予并坦然接受建设性反馈
* 承担责任，向受影响者道歉，并从经历中学习
* 不仅关注个人利益，也关注整个社区的共同利益

不可接受行为的示例包括：

* 使用性化的语言或图像，以及任何形式的性关注或性暗示
* 挑衅、侮辱或贬损性言论，以及人身或政治攻击
* 公开或私下骚扰
* 未经明确许可发布他人的私人信息（如物理地址或电子邮箱）
* 在专业场合中可合理视为不适当的其他行为

#### 执行职责

社区负责人负责阐明并执行可接受行为标准，并对任何其认为不当、威胁性、冒犯性或有害的行为采取适当且公平的纠正措施。

社区负责人有权并负有责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、维基编辑、议题及其他贡献，并在适当时说明审核决定的原因。

#### 适用范围

本行为准则适用于所有社区空间；当个人以官方身份在公共空间代表本社区时，同样适用。代表本社区的示例包括：使用官方电子邮箱、通过官方社交媒体账号发布内容，或在线上/线下活动中担任指定代表。

#### 执行

若发生辱骂、骚扰或其他不可接受行为，可向负责执行的社区负责人举报：**helloworld@linkerbot.cn**。所有投诉将及时、公正地受理与调查。

所有社区负责人均有义务尊重事件举报人的隐私与安全。

#### 执行指引

在认定任何行为违反本行为准则并决定后果时，社区负责人将遵循以下「社区影响」指引：

##### 1. 纠正（Correction）

**社区影响**：使用不当语言，或其他被视为不专业或不受欢迎的行为。

**后果**：由社区负责人发出私下书面警告，说明违规性质及该行为不当的原因。可要求公开道歉。

##### 2. 警告（Warning）

**社区影响**：单次事件或一系列行为构成的违规。

**后果**：发出警告并说明若继续该行为的后果。在指定时间内不得与涉事人员互动，包括不得主动联系执行本行为准则的人员；涵盖在社区空间及社交媒体等外部渠道中的互动。违反上述条件可能导致临时或永久封禁。

##### 3. 临时封禁（Temporary Ban）

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内禁止与社区进行任何形式的互动或公开沟通。在此期间不得与涉事人员进行公开或私下互动，包括不得主动联系执行本行为准则的人员。违反上述条件可能导致永久封禁。

##### 4. 永久封禁（Permanent Ban）

**社区影响**：表现出违反社区标准的模式，包括持续不当行为、骚扰个人，或对某类群体表现出敌意或贬损。

**后果**：永久禁止在社区内进行任何形式的公开互动。
